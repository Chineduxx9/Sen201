# Digital Clock & Stopwatch Web Application

## Project Description

The **Digital Clock & Stopwatch Web Application** is a simple and professional web-based tool developed using **HTML, CSS, and JavaScript**.  
It provides a real-time digital clock and a functional stopwatch with Start, Stop, and Reset buttons. The project emphasizes clean UI design and interactive functionality suitable for academic learning and web development practice.

---

## Project Objectives

- To display the current time in `HH:MM:SS` format using JavaScript.
- To create a stopwatch that tracks elapsed time with Start, Stop, and Reset functionality.
- To implement a responsive and professional user interface using CSS.
- To demonstrate structured web development using HTML, CSS, and JavaScript.

---

## Features

- **Digital Clock:** Updates every second to show the current time.
- **Stopwatch:** Start, Stop, and Reset controls to track elapsed time.
- **Error-free operation:** Robust handling of stopwatch state.
- **Clean UI:** Professional design with distinct sections for clock and stopwatch.
- **Single-file implementation:** All code is contained in `index.html`.

---

## Technologies Used

- **HTML5** – Structure of the application
- **CSS3** – Styling and layout
- **JavaScript (ES6)** – Logic for clock and stopwatch

## User Interface Overview

- **Clock Container (`.clock-container`)** – Displays current time.
- **Clock Display (`#clockDisplay`)** – Shows `HH:MM:SS` format.
- **Stopwatch Container (`.stopwatch-container`)** – Contains stopwatch controls and display.
- **Stopwatch Display (`#stopwatchDisplay`)** – Shows elapsed time in `HH:MM:SS`.
- **Buttons (`#startBtn`, `#stopBtn`, `#resetBtn`)** – Control stopwatch operations.
- **Author Info (`.author-info`)** – Displays the student’s name, matric number, and department.
